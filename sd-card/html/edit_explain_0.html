Welcome to the Setup of the Digitizer

This is the first time you started the digitizer after the initial installation. You have been automatically routed to the initial setup procedure. Here you adjust the settings for your counter within five steps. In the final step the inital setup will be disabled and it will restart to the normal mode. All settings will also be accessible there.

You can navigate forward and backward during the setup with the buttons "Next" and "Previous".

Do not forget to save in each step!

This is an overview over the five steps:

  1. Create reference image
    Base for the position referencing and the identification of the digits and counters.
  2. Define two unique references
    Used to align the individual camera shot and identify the absolut positions
  3. Define the digits
    Digital digits to be recognized
  4. Define the analog counters
    Analog counters to be identified
  5. General settings
    Most settings can be used with default value at the beginning. MQTT connection can be specified here

After step 5 setup mode is completed, it then reboots and starts into the normal mode.

If you need help. have a look on the Wiki and the Discussion pages.

Have fun with the digitizer!
