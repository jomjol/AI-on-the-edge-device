# Parameter `CamWbMode`
Default Value: `auto`

!!! Warning
    After changing this parameter you need to update your reference image and alignment markers!

!!! Note
    This parameter can also be set on the Reference Image configuration page!

!!! Note
    This is used when **CamAwb** is off.

**White Balance Mode**

Available options:

- `auto`
- `sunny`
- `cloudy`
- `office`
- `home`
